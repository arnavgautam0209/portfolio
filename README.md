# Arnav Gautam — Portfolio

A personal developer portfolio built with HTML, CSS, and JavaScript. Showcases projects, skills, experience, and a downloadable resume.

## 🚀 Live Demo

> Deployed via [Vercel](https://vercel.com)

## 📂 Project Structure

```
Portfolio/
├── index.html          # Home / Hero page
├── about.html          # About me & skills
├── projects.html       # Project showcase
├── experience.html     # Work & education experience
├── contact.html        # Contact form
├── resume.html         # Embedded resume viewer
├── css/
│   └── style.css       # Global styles & design system
├── js/
│   └── main.js         # Interactivity & animations
└── assets/
    ├── avatar.png          # Profile photo
    ├── resume-source.html  # Resume HTML source
    ├── resume-style.css    # Resume stylesheet
    ├── cv-page1.png        # Resume page 1 image
    └── cv-page2.png        # Resume page 2 image
```

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, flexbox, grid, animations
- **JavaScript (Vanilla)** — DOM manipulation, scroll effects
- **Google Fonts** — Inter, Outfit

## ✨ Features

- 🌙 Dark mode design with glassmorphism UI
- ⚡ Smooth scroll animations and micro-interactions
- 📱 Fully responsive (mobile, tablet, desktop)
- 🧑‍💻 Dedicated pages for Projects, Experience, and Resume
- 📬 Contact section with social links

## 🔧 Running Locally

No build step needed — just open `index.html` in your browser:

```bash
# Using Python's built-in server
python3 -m http.server 8080

# Then open: http://localhost:8080
```
